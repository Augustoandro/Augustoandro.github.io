Projects

An Intermediate Security Service Protection for Cloud Service Protection

It is a proxy server that protects cloud storage services from malware infection.
The application provides a user friendly web interface to sign up and login to platform, after which one can add his or her cloud storage bucket and credentials so that all traffic and files headed for the bucket
moves through the proxy server, where it is scanned and if found safe, it is forwarded to the bucket, otherwise the file is deleted and the user with the file uploader application gets a notification saying "The file has been found unsuitable for upload". 

Webscrap: A PHP based Search Engine with Crawler

It is basic PHP based search engine that provides results that are most similar to the ones entered by the user. The web page can filter results as images, news articles or videos. And also show all of them at the same time.
The system comes with a polite PHP crawler that crawls the web and indexes the links and websites into its database. An admin panel is used start or stop the operation of the crawler. Another PHP script runs in the background, making sure that the links indexed by the crawler haven’t turned into dead ones because of the site moving to a new address or closing down. This ensures that the results provided by the search engine are working links.

MeRiT: Media Rights Tracking using Hyperledger Fabric and AWS Rekognition

This a solution that crawls the surface and the deep web, Telegram channels, indexes it in Hyperledger Fabric, and then verifies whether the indexed media is intellectual property of someone else by comparing media over the internet using AWS Rekognition.
The solution is focused on visual media, namely images and videos. The solution is for intellectual property owners, who would be alerted if their content is available on a website without their permission.
